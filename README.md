# Dashboard de desempeño operativo.

Una planta de manufactura operaba en diversas ubicaciones internacionales (Japón, Canadá, China, Alemania, India, Estados Unidos, Francia y Brasil), generando un volumen anual de piezas, volumen anual de piezas rechazadas, un indicador de calidad y la productividad global evidenciando oportunidades de mejora en la utilización eficiente del tiempo productivo y la reducción de tiempos no productivos de forma general o mediante algun operador o mes en especifico.

El objetivo del dashboard es medir el desempeño operativo para Incrementar la productividad operativa mediante la optimización de las horas productivas en todas las ubicaciones, manteniendo el estándar de calidad superior al 99% y reduciendo el volumen de piezas rechazadas para maximizar la eficiencia del proceso productivo.

# Herramientas y tecnologías.

- Excel
- Power BI
- Lenguaje DAX
- Visualización de datos
- Optimización de procesos

# Función del Dashboard.

- Monitoreo mensual de la producción para identificar variaciones en el desempeño.
- Análisis de las horas productivas y no productivas por ubicación para identificar plantas con mayores oportunidades de mejora.
- Seguimiento mediante indicadores clave (KPIs) de producción, calidad y productividad en tiempo real.
- Filtrado de información por nombre de operador general o mensual.
